## PHASE 3: AMAZON LAUNCH (Week 10)

### Amazon Listing Optimization

**Title Formula:**
```
[Brand] + [Product Type] + [Key Feature] + [Size Range] + [Material]

Example:
Lumière Curves Plus Size Full Coverage Bra for Women | Wireless
Comfort | Sizes 36D-46G | Microfiber
```

**Bullet Points (5):**
1. SIZE INCLUSIVE: Designed specifically for plus-size bodies...
2. ALL-DAY COMFORT: Wire-free design with 4-way stretch...
3. PERFECT FIT: Wide, cushioned straps that won't dig...
4. PREMIUM QUALITY: Silky-soft microfiber fabric...
5. CONFIDENCE GUARANTEE: 30-day hassle-free returns...

**A+ Content (Brand Registry Required):**
- Brand story module
- Comparison chart
- Size guide visual
- Lifestyle imagery
- About the founders

### Amazon PPC Strategy

**Campaign Structure:**

**Campaign 1: Auto Campaign**
- Budget: $20/day
- Let Amazon find keywords
- Run for 2 weeks, harvest winners

**Campaign 2: Manual - Exact Match**
- Budget: $30/day
- Keywords from research:
  - plus size bra
  - plus size lingerie
  - comfortable plus size bra
  - wireless plus size bra
  - 42ddd bra
  - 44g bra

**Campaign 3: Product Targeting**
- Budget: $20/day
- Target competitor ASINs
- Target complementary products

### Amazon Launch Tactics

**Week 1:**
- Vine program enrollment (if eligible)
- Lightning deal (if inventory allows)
- 15% coupon visible on listing
- External traffic from email + social

**Week 2-4:**
- Monitor reviews, respond quickly
- Adjust PPC based on data
- Request reviews via Buyer-Seller messaging
- A/B test images and titles

## PHASE 5: SCALING (Months 6-12)

### Wholesale/B2B Strategy

**Target Retailers:**
- Plus-size boutiques
- Lingerie specialty stores
- Department stores (longer term)

**Wholesale Terms:**
- Minimum order: $500
- Wholesale discount: 50% off retail
- Payment: Net 30 (after relationship established)
- Lookbook + line sheets required

### International Expansion

**Priority Markets (after US):**
1. United Kingdom (similar demographics)
2. Canada (easy logistics)
3. Australia (high AOV market)
4. Germany (largest EU plus-size market)

**Considerations:**
- Currency conversion
- Shipping partners (DHL, FedEx)
- Returns logistics
- VAT/duties transparency

### Loyalty Program

**Lavena Insiders:**
- 1 point per $1 spent
- 100 points = $10 reward
- Birthday reward (15% off)
- Early access to new drops
- Exclusive colorways
