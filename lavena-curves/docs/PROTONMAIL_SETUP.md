## STEP-BY-STEP SETUP

### Step 1: Buy Domain First (Namecheap)

1. Go to [namecheap.com](https://namecheap.com)
2. Search: `lumierecurves.shop`
3. Buy domain only (no hosting needed): ~$2.98/year
4. Complete purchase

### Step 2: Create ProtonMail Account

1. Go to [proton.me/mail](https://proton.me/mail)
2. Click "Create Account"
3. Choose **Mail Essentials** plan ($6.99/month)
4. Sign up with: `ashjadmm@gmail.com` (for account recovery)
5. Create ProtonMail username: `lumierecurves@proton.me` (backup)
6. Set strong password
7. Complete payment

### Step 3: Add Custom Domain

1. Log into ProtonMail
2. Go to **Settings** → **Domain Names**
3. Click **"Add Domain"**
4. Enter: `lumierecurves.shop`
5. Click **"Next"**

### Step 4: Verify Domain (Add DNS Records)

ProtonMail will give you DNS records to add. Go to Namecheap:

1. Log into Namecheap
2. Go to **Domain List** → **lumierecurves.shop** → **Advanced DNS**
3. Add these records (ProtonMail will provide exact values):

**Verification Record:**
| Type | Host | Value |
|------|------|-------|
| TXT | @ | protonmail-verification=xxxx |

**MX Records (for receiving email):**
| Type | Host | Value | Priority |
|------|------|-------|----------|
| MX | @ | mail.protonmail.ch | 10 |
| MX | @ | mailsec.protonmail.ch | 20 |

**SPF Record (anti-spam):**
| Type | Host | Value |
|------|------|-------|
| TXT | @ | v=spf1 include:_spf.protonmail.ch mx ~all |

**DKIM Records (email authentication):**
| Type | Host | Value |
|------|------|-------|
| CNAME | protonmail._domainkey | protonmail.domainkey.xxxxx.domains.proton.ch |
| CNAME | protonmail2._domainkey | protonmail2.domainkey.xxxxx.domains.proton.ch |
| CNAME | protonmail3._domainkey | protonmail3.domainkey.xxxxx.domains.proton.ch |

**DMARC Record:**
| Type | Host | Value |
|------|------|-------|
| TXT | _dmarc | v=DMARC1; p=none |

4. Wait 15-30 minutes for DNS propagation
5. Click **"Verify"** in ProtonMail

### Step 5: Create Email Address

1. In ProtonMail, go to **Settings** → **Addresses**
2. Click **"Add Address"**
3. Create: `ashjad@lumierecurves.shop`
4. Set as **default sending address**

## EMAIL FORWARDING TO GMAIL

### Option A: Forward to Gmail (Read in Gmail)

1. In ProtonMail, go to **Settings** → **Filters**
2. Create new filter:
   - **Conditions:** All incoming mail
   - **Actions:** Forward to `ashjadmm@gmail.com`
3. Save

**Note:** ProtonMail recommends using their app instead of forwarding for full encryption.

### Option B: Use ProtonMail App (Recommended)

- Download Proton Mail app (iOS/Android)
- Add both accounts (Proton + Gmail)
- Manage all email in one place

## GMAIL "SEND AS" WITH PROTONMAIL

**⚠️ ProtonMail doesn't support SMTP for external clients by default.**

### Workaround Options:

**Option 1: Proton Mail Bridge (Desktop)**
1. Download [Proton Mail Bridge](https://proton.me/mail/bridge)
2. Install on your computer
3. It creates local SMTP/IMAP access
4. Use those credentials in Gmail "Send As"

**Bridge SMTP Settings:**
| Setting | Value |
|---------|-------|
| SMTP Server | 127.0.0.1 |
| Port | 1025 |
| Username | ashjad@lumierecurves.shop |
| Password | [Generated by Bridge] |

**Option 2: Reply from ProtonMail**
- Just use ProtonMail web/app to send as `ashjad@lumierecurves.shop`
- Use Gmail for personal, ProtonMail for business

**Option 3: Use Namecheap Email Instead**
- Simpler Gmail integration
- Lower cost
- Standard (non-encrypted) email
